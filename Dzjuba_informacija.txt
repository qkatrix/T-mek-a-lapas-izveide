# HTML Structure

HTML (HyperText Markup Language) is the standard markup language for creating web pages. The basic structure of an HTML document includes:

- **DOCTYPE declaration**: This tells the web browser which version of HTML to expect. Example: `<!DOCTYPE html>`.
- **HTML element**: The root element that wraps all other content. Example: `<html></html>`.
- **Head section**: Contains meta-information, title, and links to external resources (like CSS and JavaScript). Example: `<head><title>Page Title</title></head>`.
- **Body section**: Contains the content of the document that will be visible to the user. Example: `<body>...</body>`.

A basic structure example:
```html
<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>
</body>
</html>
```

# CSS Linking Methods

CSS (Cascading Style Sheets) can be linked to HTML documents in three main ways:

1. **Inline Styles**: Add styles directly within an HTML element using the `style` attribute. Example: `<h1 style="color:blue;">This is a heading</h1>`.
2. **Internal Styles**: Insert CSS within the `<style>` tags in the `<head>` section of the HTML document. Example:
   ```html
   <style>
       body { background-color: lightblue; }
   </style>
   ```
3. **External Styles**: Link to an external CSS file using a `<link>` tag in the `<head>` section. Example:
   ```html
   <link rel="stylesheet" type="text/css" href="styles.css">
   ```

# JavaScript Integration Approaches

JavaScript can be integrated into HTML documents in several ways:

1. **Inline Scripts**: Place JavaScript code directly within an HTML element using the `onclick` attribute or similar. Example:
   ```html
   <button onclick="alert('Hello World!')">Click me</button>
   ```
2. **Internal Scripts**: Include JavaScript in the `<script>` tags within the HTML document, typically placed before the closing `<body>` tag. Example:
   ```html
   <script>
       console.log('External JS Loaded');
   </script>
   ```
3. **External Scripts**: Link to an external JavaScript file using the `<script>` tag in the `<head>` or just before the closing `<body>` tag. Example:
   ```html
   <script src="script.js"></script>
   ```

In summary, HTML provides structure, CSS controls presentation, and JavaScript adds behavior to web pages.
