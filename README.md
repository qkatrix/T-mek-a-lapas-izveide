# Web Page Basics Learning Project

**Author**: Jekaterina Dzjuba, 12.b Class  

## Project Description
This project serves as a fundamental introduction to creating web pages. It covers essential concepts of HTML, CSS, and JavaScript, providing learners with practical examples and exercises.

## File Structure
```
/ 
|-- index.html       # Main HTML file
|-- styles/          # Directory for CSS files
|   |-- style.css    # Main CSS file
|-- scripts/         # Directory for JavaScript files
|   |-- app.js       # Main JavaScript file
|-- images/          # Directory for images used in the project
|-- README.md        # Project documentation
```

## Features
- Basic HTML structure
- CSS for styling
- JavaScript for interactivity
- Responsive design techniques

## Usage Instructions
1. Clone the repository:
   ```
   git clone https://github.com/qkatrix/T-mek-a-lapas-izveide.git
   ```
